# simple_RESTful_API_nodejs_MCR
This is a simple RESTful API using NodeJs with 3 packages :
<ul>
  <li>Express</li>
  <li>MySQL</li>
  <li>Body-parser</li>
  <li>Dotenv</li>
  <li>Bcrypt</li>
  <li>Jsonwebtoken</li>
</ul>

How to run :
<pre>node app.js</pre>
Or you can use nodemon:
<pre>npm run dev</pre>
